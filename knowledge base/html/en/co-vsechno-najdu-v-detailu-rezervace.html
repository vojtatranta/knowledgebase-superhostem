Reservation management and calendar

What can I find in the reservation details?

Reservation detail is the workspace for one specific stay. You can verify data, copy the guest message, check submitted forms, and quickly see the next step.

Help Center Library Next article →

When to open reservation detail

  • when you want to verify that the reservation belongs to the right property and dates
  • when you need to edit guest name, guest count, guest language, or price
  • when you want to send the guest a registration link or check the message text
  • when you need to check whether guests completed the domovní kniha form
  • when you need to see which emails the system sent for this reservation
  • when you need to handle Ubyport reporting for foreign guests
  • when you want to create an invoice or check the reservation price

What to check at the top

The top part gives you the quick reservation summary: guest, check-in, check-out, relative time to arrival, and property name. It also includes the action for syncing the reservation source.

  1. verify guest and property
  2. check check-in and check-out dates
  3. check whether the reservation is cancelled
  4. if data does not match, use source sync
If the system finds missing profile or property data, it shows a warning above the content. These fields are usually needed for reports, exports, or follow-up administration.

How to read the reservation workflow

The workflow shows the current phase of the reservation and who needs to act next.

  • Reservation import means the reservation already exists in Superhostem.
  • Prepare guest message is the host step where you copy or edit the message with the link.
  • Guest registration means the guest is filling in the form or has already completed it.
  • Foreign police report appears mainly for foreign guests.
  • Completed means there is no regular next action for this reservation.

Do not read workflow as decoration. It answers: What should I do with this reservation now?

Overview: guest, dates, price, source

The Overview tab contains two main cards.

In Guest information, you can check or edit:

  • guest name
  • email and phone, if available
  • number of guests
  • guest language for messages and forms

Guest count matters. If it is set to 0, the system warns you because registration links and complete domovní kniha data depend on the correct guest count.

In Reservation details, you check:

  • check-in date and time
  • check-out date and time
  • reservation source, such as Airbnb, Booking.com, or manual entry
  • link to the original platform reservation, if available
  • total reservation price

You can edit the price directly in detail. This helps when iCal brought dates, but financial data arrived later through email forwarding or must be entered manually.

Guest message and registration links

When the reservation waits for guest data, detail shows the next steps. This is where you prepare the message for the guest.

  • guest message preview
  • copy message button
  • option to edit message text for this reservation
  • reservation confirmation link
  • registration form link after confirmation

Link types:

  • confirmation link helps the guest confirm the reservation and continue to the next step
  • registration form collects guest data for domovní kniha
  • property permalink fits Airbnb automated messages so guests can find their reservations for that property

If the property has automated message templates, detail can also show their preview for the selected guest language.

You do not have to use only the default message text. For each property, you can prepare your own message templates and then check in reservation detail how the specific guest, dates, price, or registration link will be filled in.

Want to send messages automatically?

If you do not want to copy the message manually for every reservation, set up message automation for the whole property. The template can then send the right link and instructions based on the specific reservation.

Registered guests

Registered guests shows how many forms are completed and what is still missing for each guest.

The table includes:

  • guest order in the reservation
  • name and email
  • date of birth
  • address
  • guest type, such as adult, child, or guest without fee
  • nationality
  • document type and document number
  • document number photo, if uploaded
  • completion status

For a completed guest, you can open the form, copy the link, edit data, or delete the guest. If the guest has not filled in data yet, you see Pending.

Ubyport and foreign guests

If the reservation contains foreign guests, detail can show the Ubyport action. It depends on data from the guest form.

Before sending to Ubyport, check:

  • guest count matches reality
  • foreign guests have nationality filled in
  • date of birth is present
  • document type and document number are present
  • stay address and property data are correct

If something is missing, fix the guest or property settings first. Export or submission makes sense only after that.

Emails

The Emails tab shows email history for the reservation.

  • when the email was sent
  • recipient
  • subject
  • email type, such as registration invite, edit link, or confirmation
  • status, such as sent, delivered, failed, or bounced

This tab helps when the guest says they did not receive anything, or when you need to verify that the system sent the message.

Fast reservation check

  1. Verify guest, property, and stay dates.
  2. Check guest count and guest language.
  3. Read workflow and decide the next step.
  4. If the guest has not filled in data yet, copy the message or registration link.
  5. After forms are submitted, check the registered guests table.
  6. For foreign guests, check Ubyport data.
  7. If the guest says no message arrived, open the Emails tab.
  8. If date, price, or source does not match, check iCal and email forwarding.

What to do if something does not match

If dates or the reservation itself are missing, start with iCal connection. If price or financial data is missing, check email forwarding. If guest data is missing, send or copy the registration link from reservation detail.

  • Does the stay match? Check dates, property, and source.
  • Do I have guest data? Check guest count and registered guests table.
  • Was communication sent? Check message and emails tab.
  • Can I report to Ubyport? Check foreign guests and document completeness.
