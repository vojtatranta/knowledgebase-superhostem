What to do when email forwarding does not work

Email forwarding starts working only after you copy the assigned address correctly, confirm it in Gmail, and create the forwarding rule.

Help Center Email forwarding Next article →

Quick check

If reservation emails are not reaching Superhostem, the issue is usually in one of three steps: the copied address, Gmail confirmation, or the forwarding rule.

  1. Check that Gmail contains the exact address assigned by Superhostem.
  2. Confirm the forwarding address in the verification email sent by Gmail.
  3. Only after that, create the filter or rule that actually forwards reservation emails.

1. The assigned address was copied incorrectly

Gmail must forward messages to the exact address from Superhostem. If one character is missing, a space is added, or an old address is used, emails will not reach Superhostem.

  • copy the address again directly from Superhostem
  • check both the start and end of the address
  • do not type the address manually, copy it instead

2. The forwarding address is not confirmed in Gmail

After adding a new forwarding address, Gmail sends a verification email. Until it is confirmed, Gmail does not forward anything to that address.

If you use multiple Gmail accounts on one computer, first make sure you are switched to the account where you are setting up forwarding.
  • open the correct Gmail account
  • find the verification email from Gmail
  • confirm the forwarding address

3. The forwarding filter or rule is missing

Confirming the address is not enough. After confirmation, create a condition that selects reservation emails and forwards them to Superhostem.

  • create the filter only after confirming the forwarding address
  • check that the filter matches emails from Airbnb, Booking.com, or another source
  • the filter must use the action to forward to the Superhostem address

If it still does not work

Send a test reservation email or wait for the next real reservation email. If it does not appear in Superhostem, check the Gmail filter history and the correct Gmail account where forwarding is configured.
