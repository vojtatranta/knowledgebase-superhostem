Automation and communication

How to automate guest messages

Set up one pre-arrival message. The guest gets the right link at the right time, and you do not send it manually for every booking.

Help Center Previous article Next article →
Final setup step

In Superhostem, you decide when the guest link becomes available. Use the same timing for the automated message in Airbnb or Booking.com.

Start with one message before arrival

Do not automate everything at the beginning. One useful message is enough: it sends the form or guestbook link before arrival.

  1. 1Copy the link
  2. 2Set availability
  3. 3Add it to the message
  4. 4Check the result

Where to start in Superhostem

Open the property detail and go to Automation.

  • copy the guest link
  • set how many days before arrival the booking should be available through the link
  • save the setting
The number of days must match the message timing in Airbnb or Booking.com. If Superhostem is set to 3 days before arrival, the automated message should also go out 3 days before arrival.

Where to place the link

  1. AirbnbMessages → message settings

    Create a quick reply or automated message. Add the Superhostem link and assign the message to the correct listing.

  2. Booking.comExtranet → message settings

    Create a message template, paste the Superhostem link, and schedule it before arrival.

Check before saving

  • the message contains the link for the correct property or unit
  • the template is assigned to the correct listing
  • the send time matches the number of days set in Superhostem
  • the text clearly tells the guest what to do after clicking
For short stays and last-minute bookings, test the timing first. The message must be sent when the booking is already available through the link.
