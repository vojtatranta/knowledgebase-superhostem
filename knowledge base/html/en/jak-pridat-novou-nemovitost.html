Quick start

How to add your property

You can create your first property without an iCal link. If you already have an iCal URL from a booking portal, use it to import reservations straight away.

Help CenterPrevious stepNext step →
Start by creating the property.

Choose the remaining settings only when they apply to the property.

Choose a route based on your iCal URL

An iCal URL can come from Airbnb, Booking.com, e-Chalupy, or another booking portal.

Two routes for creating a property

  1. You do not have an iCal URL yetCreate the property from Properties

    Open Properties in the left menu, select Add property, then continue to its settings.

    Add property →
  2. You already have an iCal URLUse onboarding and import

    On the Overview page, enter the property name and iCal URL. Then select Create property and import reservations.

    Continue with iCal import →
