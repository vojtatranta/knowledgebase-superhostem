Properties and calendars

How to share a web calendar by link

Use a web calendar link for people who need to see occupancy but should not have access to your Superhostem account. Typical use cases are cleaners, property managers, owners, or external partners.

Help Center Previous article Next article →
When to use it

Use it when someone only needs to open a browser link and see dates. They do not need a calendar app setup or a Superhostem user account. It also works well for people who only need temporary calendar access.

What the shared web calendar is for

You create a unique calendar link. Anyone with the link sees the web occupancy view with only the details you allow.

  • cleaners see when the property needs to be ready
  • a manager sees occupancy without admin access
  • an owner or partner can check reservations in a browser
  • you can revoke an old link at any time

When details are missing in the calendar

If guest name, guest count, price, or other details are missing in the calendar, Superhostem does not have that data yet. Enable reservation email synchronization to fill these details quickly. Without it, the details must be entered manually by the guest or by you in the admin.

Security

  • Anyone with the link can open it.
  • Do not post the link in public groups or public documents.
  • Share only the fields external people truly need.
  • Regularly revoke old or unused links.

Difference from a calendar app

The web calendar is a normal browser link. The recipient does not need Apple Calendar, Google Calendar, or Outlook. A calendar app connection is better when someone wants the dates inside their personal calendar.
